# design.md — Piscine 42 Exam Simulator (web)

8-bit sobre, dark, franc, anglais, très peu de texte. Arcade / CRT terminal.
Couleurs (zéro jaune, zéro rose pâle) : bg #0b0d12, green #5bffa0 (PASS), red #ff5c6c (FAIL),
ramp niveaux cyan->bleu->lilas->orange->rouge. Typo self-hosted : Press Start 2P (titres),
VT323 (terminal), Departure Mono (éditeur). Wordmark PISCINE.EXE (pas de logo slop).
Compilateur intégré = Wandbox API client-side (gcc -std=gnu17 -Wall -Wextra -Werror).
